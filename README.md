# 🌦️ AI Weather Data Pipeline

This project is an end-to-end weather data pipeline built using modern Data Engineering and Machine Learning tools.

The pipeline automatically collects weather data from an API, stores it in a database, trains a machine learning model and predicts future temperatures.

---

## 🚀 Technologies Used

- Apache Airflow
- Docker
- PostgreSQL
- Scikit-learn
- Metabase
- Python
- Git

---

## 🏗️ Architecture

Weather API  
↓  
Apache Airflow Pipeline  
↓  
PostgreSQL Database  
↓  
Machine Learning Model (Temperature Prediction)  
↓  
Metabase Dashboard

---

## ⚙️ Pipeline Steps

1. Fetch weather data from API
2. Store data in PostgreSQL
3. Train machine learning model
4. Predict future temperature
5. Visualize results in Metabase

---

## 📊 Features

✔ Automated data pipeline using Apache Airflow  
✔ Weather data storage in PostgreSQL  
✔ Machine learning model for temperature prediction  
✔ Data visualization dashboard using Metabase  
✔ Containerized environment with Docker
